# template-copy

テンプレートファイルをコピーして，日付付きの新しいファイルを自動作成する Windows 用バッチです．

作成されるファイル名は，次の形式です．

`YYYYMMDD-プロジェクト名-作成者名.pptx`

同名ファイルが存在する場合は，連番付きのファイル名を提案します．
上書きは行いません．
作成後は，生成したファイルを自動で開きます．

## できること

* テンプレートファイルをコピーして新規作成
* ファイル名を日付付きで自動生成
* 同名ファイルがある場合は `_2`，`_3` のように連番を付与
* 上書き防止
* 初回起動時に設定ファイルを自動生成
* 作成後にファイルを自動で開く

## 動作環境

* Windows
* PowerShell が利用可能な環境

## 使い方

1．`create_from_template.bat` を配置します．
2．テンプレートファイル（例: `tmp.pptx`）を同じフォルダに置きます．
3．`create_from_template.bat` をダブルクリックで実行します．
4．初回のみ，テンプレートファイル名と作成者名を入力します．
5．プロジェクト名を入力すると，新しいファイルが作成されます．

## 設定ファイル

初回実行時に `settings.ini` が自動生成されます．

例：
```ini
TEMPLATE_NAME=tmp.pptx
AUTHOR_NAME=YourName
```

## 注意

* `settings.ini` は各環境ごとの設定ファイルです．Git 管理から除外することをおすすめします．
* ファイル名に使えない文字（`\ / : * ? " < > |`）は入力できません．
